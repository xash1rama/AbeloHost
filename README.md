###AbeloHost 
Приложение на Python 3.12.

🚀 Как запустить
1. Клонирование и подготовка
Вытащите проект из репозитория и перейдите в папку (находись в консоли там куда будешь тащить приложение):

git clone git@github.com:xash1rama/AbeloHost.git

2. Запуск Docker
Перейдите в папку с приложением и введите команду в консоли:
`docker-compose up --build`

3. Проверка
Зайди в браузере на `http://localhost:8000/docs`


#🛠 Как тестировать
1. Находится в нужном каталоге
Нужно находиться в рабочей дериктории "BOTO_ED/boto_test"

2. Запуск тестов
Когда контейнеры запущены:
Ввести команду:
`docker exec -it abelo_fastapi env PYTHONPATH=. pytest -v`
